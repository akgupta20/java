variable name can start with '$','_' and any alphabet.

Types: 
1. Primitive:

char: 2 Byte

byte: 1 Byte

short: 2 Byte

int: 4 Byte

long: 8 Byte > eg. long a = 100l; # add l in last to consider as long

These above are integral types means only integers are assignable.


float => 4 byte IEEE 754 representation => eg. float a = 0.1f; # add f as suffix

double => 8 byte IEEE 754 representation



boolean => 1 Bit
